# Changelog

## 0.2.0 — 2026-07-12

- 修复 Tauri `beforeDevCommand` / `beforeBuildCommand` 递归调用导致的启动与构建问题。
- 对工作流、模型、会话等动态内容统一进行 HTML 转义，避免界面注入。
- Web 预览缺少 Tauri 原生桥接时显示可理解的桌面端提示，不再暴露底层异常文本。
- 改进设置校验、编辑器删除确认、运行状态恢复与聊天完整回复回退。
- 对流式日志按动画帧批量刷新并限制缓存长度，降低长任务界面卡顿。
- 完善键盘焦点、ARIA 状态、窄窗口布局和减少动态效果偏好支持。
- 前端构建新增严格 TypeScript 检查，并移除未使用的 `js-yaml` 依赖。
- 升级到 Vite 6.4.3，消除 Vite 5.4.21 的已知高危与中危漏洞。
- CLI 改为无 shell 的跨平台进程启动，并严格校验 Rust target triple，关闭命令注入面。
- API Key 从明文 JSON 迁移到操作系统凭据库；旧版密钥会在启动时迁移并擦除明文副本。
- LLM 请求仅允许官方 Ark HTTPS 域名和协议对应路径，同时禁止 HTTP 重定向携带 Bearer Token。
- 移除仓库内硬编码的本机 Windows 签名证书指纹，发布签名改由发布环境显式配置。
