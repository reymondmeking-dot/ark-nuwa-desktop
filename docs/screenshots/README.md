# 应用截图

本目录存放应用截图，用于 README 和文档展示。

## 截图列表

| 文件名 | 说明 | 建议尺寸 |
|--------|------|----------|
| `settings.png` | 设置页 - Ark 配置与连接测试 | 1200x800 |
| `runner.png` | 运行页 - Workflow 执行与可视化 | 1200x800 |
| `editor.png` | 编辑器页 - YAML 工作流编辑与验证 | 1200x800 |
| `chat.png` | 对话页 - 蒸馏后智能体对话 | 1200x800 |

## 截图指南

1. 运行 `pnpm tauri dev` 启动应用
2. 调整窗口到合适大小（建议 1200x800）
3. 截图保存为 PNG 格式
4. 放入本目录后更新 README.md 中的图片链接
